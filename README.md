Cybersecurity Awareness Chatbot
A simple C# console-based chatbot that teaches people basic cybersecurity in a fun, interactive way.
What it does

Greets you with voice (text-to-speech)
Shows a cool ASCII art logo
Talks to you like a real person
Covers important topics like strong passwords, spotting phishing attempts, and safe browsing habits
Validates your inputs so you don’t break it
Uses nice colors and a typing effect to feel more alive

Technologies Used

C# (.NET)
GitHub
GitHub Actions (for CI)

How to Run

Open the project in Visual Studio
Press F5 (or click Run)
Just follow the prompts and chat with it
